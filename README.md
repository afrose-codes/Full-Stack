# Full Stack Projects

## 📌 Task 1: Student Registration System

### 🔹 Description
This project is a simple Student Registration System where users can enter student details and store them in a database.

### 🔹 Features
- Register student details (Name, Email, DOB, Department, Phone)
- Store data in MySQL database
- View all student records

### 🔹 Technologies Used
- HTML5
- CSS3
- PHP
- MySQL
- XAMPP

 ## 🚀 How to Run
1. Start XAMPP (Apache + MySQL)
2. Place project folder inside `htdocs`
3. Open browser and go to:

```
http://localhost/student_project/Task1_Student_Registration/index.html
```

### 🔹 Output
- Registration form
- Data stored in database
- Display of student records
